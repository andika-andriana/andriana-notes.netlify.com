Persamaan || FOR - forEach - MAP

Persamaan antara for, forEach dan MAP adalah sama sama untuk perulangan dan juga bisa melakukan perulangan berdasarkan data pajang dari sebuah array. berikut penjelasannya:

  • for ( ) adalah fungsi perulangan yang disediakan oleh javascript, berikut adalah strukstur pembuatan fungsi for:
    function perulangan() {
  for (i = 0; i < 10; i++) { //memulai perulangan
    console.log("Jalan: " + i);
  }
}
perulangan()

    Struktur for:

    • for ( i = 0; ‚Üí pendeklarasian variabel i yang awal mula angkanya adalah 0
    • i < 10; ‚Üí ketentuan yang harus terpenuhi untuk melakukan perulangan
    • i ++) ‚Üí action yang dilakukan jika kondisi terpenuhi yaitu variable i akan ditambah 1 setiap kali perulangan dilakukan

  • forEach( ) ‚Üí untuk melakukan perulangan khusus pada nilai dalam variabel Array.
    • Contoh penerapannya:
    function forEachArry() {
  var arry = ["Budi", "Doni", "Boby"];

  arry.forEach(function(item, index, array) {
    console.log("Itemnya apa: ", item);
    console.log("Index ke berapa: ", index);
    console.log("Dari Array Mana: ", array);
  })
}

  var arry = ["Budi", "Doni", "Boby"];

  • MAP ( ) ‚Üí Method ini hampir sama dengan method forEach yaitu berfungsi untuk melakukan perulangan pada nilai dalam variabel array.

    Namun Method ini memiliki perbedaan baik dengan forEach maupun dengan for, Tetapi perbedaan tersebut akan disampaikan dalam materi selanjutnya:

    • Berikut contoh penerapannya:
    function mapArry() {
  var arry = ["Budi", "Doni", "Boby"];

  arry.map((item, index, array) => {
    console.log("Itemnya apa: ", item);
    console.log("Index ke berapa: ", index);
    console.log("dari data array: ", array);
  })
}
