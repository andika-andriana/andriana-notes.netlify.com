Perulangan Bersarang (Nested Loop)

Perulangan Bersarang adalah perulangan dalam perulangan, hal ini bisa dilakukan pada javascript.

Berikut cara melakukannya:

for (i = 0; i < 2; i++) {
  console.log("Perulangan I sampai nilai ", i);
  for (j = 0; j < 2; j++) {
    console.log("Perulangan J sampai nilai ", j);
  }
  console.log("---Percobaan Perulangan ke---", i);
}

maka outputnya adalah:

PS C:\Users\AndikaAndriana\Documents\1-repo-belajar\javascript> node .\nested-loop.js
Perulangan I sampai nilai  0
Perulangan J sampai nilai  0
Perulangan J sampai nilai  1
---Percobaan Perulangan ke--- 0
Perulangan I sampai nilai  1
Perulangan J sampai nilai  0
Perulangan J sampai nilai  1
---Percobaan Perulangan ke--- 1

Penjelasan pada notes koding berikut :

for (i = 0; i < 2; i++) { //perulangan i 
  console.log("Perulangan I sampai nilai ", i); //kemudian mencetak hasilnya
  for (j = 0; j < 2; j++) { //kemudian perulangan dilanjutkan oleh j
    console.log("Perulangan J sampai nilai ", j); //akan terus dilakukan sampai j < 2
  } //setelah hasil terpenuhi perulangan kembali ke i untuk melanjutkan perulangan
  console.log("---Percobaan Perulangan ke---", i); //diberikan untuk memberi garis baru
}

itulah mengapa j lebih banyak memberikan output dibanding i karena j akan melakukan perulangan sampai nilai j terpenuhi (lihat pada output dibawah):
