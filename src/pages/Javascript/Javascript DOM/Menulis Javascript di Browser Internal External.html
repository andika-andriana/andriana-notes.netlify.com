Menulis Javascript di Browser (Internal & External)

Menuliskan Javascript di Browser

Untuk menuliskan javascript di browser bisa dilakukan dengan 2 cara, yaitu:

  1. Internal

    Kita akan coba menuliskan code javascript di browser secara internal.

    Pertama, buat dulu 1 buah file html, contoh index.html

    <!DOCTYPE html>
<html>
 <head>
   <meta charset="utf-8">
   <title>Belajar JavaScript</title>
 </head>
 <body>
   <h1>Belajar JavaScript DOM</h1>
 </body>
</html>

    Untuk menuliskan code javascript secara internal di dalam file html adalah dengan menambahkan element script seperti dibawah ini:

    <script type="text/javascript">
</script>

    Code tersebut diletakkan diatas tag penutup </body>, sehingga code HTML kita akan menjadi seperti ini:

    <!DOCTYPE html>
<html>
 <head>
   <meta charset="utf-8">
   <title>Belajar JavaScript</title>
 </head>
 <body>
   <h1>Belajar JavaScript DOM</h1>

   <script type="text/javascript">
     
   </script>
 </body>
</html>

    Kemudian kita akan menuliskan code javascript, contoh : console.log();

    <!DOCTYPE html>
<html>
 <head>
   <meta charset="utf-8">
   <title>Belajar JavaScript</title>
 </head>
 <body>
   <h1>Belajar JavaScript DOM</h1>

   <script type="text/javascript">
     console.log("Menuliskan Code Javascript di Browser Secara Internal");
   </script>
 </body>
</html>
  1. Eksternal

    Kita akan coba menuliskan code javascript di browser secara eksternal.

    Pertama, buat dulu 1 buah file html contoh index.html dan 1 buah file javascript contoh index.js

    Tuliskan code pada index.html seperti dibawah ini :

    <!DOCTYPE html>
<html>
 <head>
   <meta charset="utf-8">
   <title>Belajar JavaScript</title>
 </head>
 <body>
   <h1>Belajar JavaScript DOM</h1>
 </body>
</html>

    Untuk menuliskan code javascript secara eksternal di dalam file html adalah dengan menambahkan element script seperti dibawah ini :

    <script type="text/javascript">
</script>

    Kemudian kita linkkan dengan file javascript kita :

    <script type="text/javascript" src="index.js"></script>

    Selanjutnya, kita akan menuliskan code javascript kita di dalam file index.js yang sudah terhubung dengan file index.html

    console.log("Menuliskan Code Javascript di Browser Secara Eksternal");

Menjalankan Javascript di Browser

Pada awalnya javascript adalah bahasa pemrograman yang hanya berjalan di sisi client (dalam hal ini browser). Sejak ada Node JS, javascript dapat berjalan di luar environment browser contohnya di komputer kita atau di server.

Kita akan coba untuk menjalankan javascript di browser, baik yang ditulis secara INTERNAL ataupun secara EKSTERNAL adalah :

  1. Pastikan sudah membuat sebuah file html dan menuliskan javascript di dalamnya (internal) ataupun menuliskan javascript di luar (eksternal), yang ada pada materi Menuliskan Javascript di Browser
  1. Buka Web Server for Chrome
  1. Pilih pada CHOOSE FOLDER dan arahkan ke dalam folder tempat kamu menuliskan code html yang sudah terhubung dengan javascript
  1. Klik URL yang terletak dibawah tulisan Web Server URL(e), biasanya linknya: http://127.0.0.1:8887 dan akan tampil seperti dibawah ini:
  1. Untuk melihat response dari code javascript yang sudah kita tulis yaitu berupa console.log, silahkan lakukan inspect elements dengan cara KLIK KANAN, dan pilih inspect
  1. Untuk melihat hasilnya silahkan klik menu CONSOLE yang berada di samping menu ELEMENTS, Maka kita akan bisa melihat hasilnya berupa text hasil console.log seperti dibawah ini :
