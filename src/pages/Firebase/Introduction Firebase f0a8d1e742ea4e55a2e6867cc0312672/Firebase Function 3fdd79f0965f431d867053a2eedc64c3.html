Firebase Function

Dengan Cloud Functions for Firebase, Kita dapat menjalankan kode backend secara otomatis sebagai respons terhadap peristiwa yang dipicu oleh fitur Firebase dan permintaan HTTPS. Kode Kita disimpan di cloud Google dan dijalankan di lingkungan yang terkelola. Kita tidak perlu mengelola atau menyesuaikan skala server sendiri.

Setelah Kita menulis dan menerapkan fungsi, server Google akan segera memulai pengelolaan fungsi. Kita dapat menjalankan fungsi secara langsung dengan permintaan HTTP, atau dalam kasus fungsi background, server Google akan mendeteksi peristiwa dan menjalankan fungsi saat dipicu.

Seiring bertambahnya atau berkurangnya beban, Google akan merespons dengan menskalakan jumlah instance server virtual yang dibutuhkan secara cepat untuk menjalankan fungsi Kita. Setiap fungsi berjalan secara terisolasi di lingkungannya sendiri dengan konfigurasinya sendiri.
