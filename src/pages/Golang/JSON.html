    Di Go, data json dituliskan sebagaiÂ string. Dengan menggunakanÂ json.Unmarshal, json string bisa dikonversi menjadi bentuk objek, entah itu dalam bentukÂ map[string]interface{}Â ataupun objek struct.

    Pertama import package yang dibutuhkan, lalu siapkan structÂ User.

    Silakan tulis kode berikut pada functi main:

    package main

import "encoding/json"
import "fmt"

type User struct {
	FullName string `json:"Name"`
	Age      int
}

func main() {
	var jsonString = `{"Name": "john wick", "Age": 27}`
	var jsonData = []byte(jsonString)

	var data User

	var err = json.Unmarshal(jsonData, &data)
	if err != nil {
		fmt.Println(err.Error())
		return
	}

	fmt.Println("user :", data.FullName)
	fmt.Println("age  :", data.Age)
}

  1. Decode Array JSON Ke Array Objek

    Decode data dari array json ke slice/array objek masih sama, siapkan saja variabel penampung hasil decode dengan tipe slice struct. Contohnya bisa dilihat pada kode berikut.

    var jsonString = `[
    {"Name": "john wick", "Age": 27},
    {"Name": "ethan hunt", "Age": 32}
]`

var data []User

var err = json.Unmarshal([]byte(jsonString), &data)
if err != nil {
    fmt.Println(err.Error())
    return
}

fmt.Println("user 1:", data[0].FullName)
fmt.Println("user 2:", data[1].FullName)

    Setelah sebelumnya dijelaskan beberapa cara decode data dari json string ke objek, sekarang kita akan belajar caraÂ encodeÂ data objek ke bentuk json string.

    FungsiÂ json.MarshalÂ digunakan untuk decoding data ke json string. Sumber data bisa berupa variabel objek cetakan struct,Â map[string]interface{}, atau slice.

    Pada contoh berikut, data slice struct dikonversi ke dalam bentuk json string. Hasil konversi berupaÂ []byte, casting terlebih dahulu ke tipeÂ stringÂ agar bisa ditampilkan bentuk json string-nya.

import "fmt"

type User struct {
	FullName string `json:"Name"`

}

func main() {
	var object = []User{{"john wick", 27}, {"ethan hunt", 32}}
	var jsonData, err = json.Marshal(object)
	if err != nil {
	    fmt.Println(err.Error())
	    return
	}
	
	var jsonString = string(jsonData)
	fmt.Println(jsonString)
}
