Perulangan Bersarang

Tak hanya seleksi kondisi yang bisa bersarang, perulangan juga bisa.

Cara pengaplikasiannya kurang lebih sama, tinggal tulis blok statement perulangan didalam perulangan, Contoh:

package main

import "fmt"

func main() {
	for i := 0; i < 5; i++ {
		for j := i; j < 5; j++ {
			fmt.Print(j, " ")
		}

		fmt.Println()
	}
}

Hasil:

PS C:\Go\src\belajarGolang> go run .\perulangan.go
0 1 2 3 4
1 2 3 4
2 3 4
3 4
4
