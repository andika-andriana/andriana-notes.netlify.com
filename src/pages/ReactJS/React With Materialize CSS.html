React With Materialize CSS

Materialize ini adalah kerangka kerja seperti bootstrap dimana framework khusus untuk frontend yang terdiri dari HTML. CSS, dan Javascript yang tinggal kita gunakan. Yang uniknya disini Materialize mengimplementasikan Material Design.

Berikut cara penerapannya pada ReactJS:

  • Buat project react baru (contoh nama project: hello-materialize):
    create-react-app hello-materialize
  • Buka project tersebut pada text editor(contoh: atom.io)
  • Hapus file bawaan yang tidak digunakan:
  • modifikasi kode yang tersambung pada file yang kita hapus tadi kode (agar tidak terjadi error):
  • Jalan kan npm untuk memastikan tidak ada yang salah:
  • Sebelum melakukan installasi framework materialize, kita cak dulu pada package.json untuk memastikan bahwa framework tersebut belum terinstall:
  • buka terminal baru pad folder project kita untuk menginstall semantic-ui:
  • Ketikan pada terminal tersebut (pastikan anda terhubung dengan internet):
    npm install --save react-materialize
  • Tunggu prosesnya hingga selesai:
  • Untuk memastikan bahwa framework sudah berhasil terinstall cek pada package.json:
  • Kemudian import semantic-ui:
    • Tambahan Pada Head Di Index.html:
      <!-- Import Google Icon Font -->
  <link href="http://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
  <!-- Import materialize.css -->
  <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/css/materialize.min.css" />
    • Tambahan Pada Body Di Index.html:
      <!-- Import jQuery before materialize.js -->
  <script src="https://code.jquery.com/jquery-2.1.1.min.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/js/materialize.min.js"></script>
  <!-- And then your bundled js -->
  <script src="path/to/your/bundle.js"></script>
  • Mari kita coba gunakan pada App.js (React):
    import React from "react";
import { Button } from "react-materialize";
function App() {
  return (
    <div>
      <Button>Test</Button>
    </div>
  );
}

export default App;
