Membuat dan Menjalankan Project React JS

Untuk membuat sebuah project dengan menggunakan React JS tentunya kita membutuhkan konfigurasi yang harus disiapkan, namun karena kita masih dalam tahap awal belajar mengenai react js dan untuk memudahkan kita dalam belajar react, kita bisa membuat sebuah project react js menggunakan package manager create-react-app

  1. Kita sudah menginstall create-react-app di komputer kita, Untuk mengeceknya kita bisa melakukannya dengan cara:
    • Buka terminal pada komputer kita ketik : create-react-app --version kemudian tekan ENTER

    [CATATAN]: Setelah kita menjalankan create-react-app --version dan kemudian version dari create-react-app sudah muncul itu artinya bahwa create-react-app sudah terinstall di komputer kita

  1. Sekarang kita akan mencoba membuat project React JS pertama kita
    • Buka Terminal dan pilih lokasi tempat untuk meletakkan project React JS nya, terserah peserta masing-masing apakah akan diletakkan dalam folder Documents ataupun ditempat yang lain
    • Kemudian Ketik create-react-app hello-react dan tekan ENTER

      Kita memberi nama project kita hello-react, dan jika ingin menggunakan nama lain selain hello-react diperbolehkan karena hello-react hanya sebuah nama dari project / folder.

    • Setelah create-react-app hello-react dan tekan ENTER kita tunggu hingga prosesnya selesai
    • Dan jika sudah selesai akan menampilkan
    • Ketikkan cd hello-react kemudian tekan ENTER Cara tersebut digunakan untuk kita masuk ke dalam folder project react js kita yang bernama hello-react
  1. Sekarang kita akan mencoba Menjalankan project React JS pertama kita.
    • Masih didalam directory / folder hello-react, Ketikkan npm start kemudian tekan ENTER Cara tersebut digunakan untuk menjalankan project react kita. Dan sekarang tunggu hingga prosesnya selesai, saat prosesnya sudah selesai akan menampilan seperti ini pada terminal kita
  1. Untuk mengecek tampilan default project react js pertama kita.
    • Silahkan buka web browser contoh Google Chrome
    • Buka http://localhost:3000 
    • Dan kita akan melihat hasilnya seperti dibawah ini
