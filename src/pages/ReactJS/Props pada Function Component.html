Props pada Function Component

Type data pada component react itu cuma dua yaitu Prop dan State.

  • Jika data berubah maka component berubah ( render ulang )
  • Ada 2 cara membuat component,¬†functional component¬†dan¬†class component\
    • Stateless Component ‚Äî tidak memiliki state hanya prop. Umumnya component ini di buat dengan functional component karena codenya lebih ringkas;
    • Statefull Component ‚Äî memiliki state & props. Component ini dibuat dengan class. Kelebihan dari class component adalah memiliki lifecycle
  • Untuk sekarang ingat saja functional component tidak memiliki state, itu sebabnya barang ini biasa disebut stateless component.
  • Sedangkan class component bisa memiliki prop dan state.

Prop singkatan dari Property adalah sebuah nilai yang bisa diberikan terhadap suatu component, sehingga component tersebut bisa lebih dinamis.

  • Prop itu read-only.
  • Prop umumnya digunakan untuk komunikasi data component dari¬†parent¬†komponent ke¬†child¬†component.

Berikut cara pembuatan props yang di buat menggunakan functional component:

  • Membuat props pada component Footer yang berada pada App.js:

import Header from './Header.js';
import Footer from './Footer.js';

function App() {
  return (
    <div>
      <Header/>
      <List/>
      <Footer name=" -- React JS --"/> //name adalah nama propsnya
    </div>
  );
}

export default App;

function Footer( props ) {

    <div>
      <hr/>

    Perubahan data pada props dan state akan merender ulang component.

    • Gunakan props untuk komunikasi data antar komponent ( parent to child component )
    • Gunakan state untuk komunikasi data internal, hanya dalam component itu sendiri
    • props dan state bisa memiliki default value
