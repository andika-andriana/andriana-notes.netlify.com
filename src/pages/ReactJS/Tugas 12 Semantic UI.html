Tugas 12 Semantic UI

Objectives

  • Memahami cara penggunaan framework css semantic ui.
  • Memahami cara penggunaan component ( Form, Checkbox ).

Hints

  • Sebelum menggunakan sebuah component silakan anda import terlebih dahulu component yang ingin digunakan.
    import {
Form,
Checkbox
} from "semantic-ui-react";
  • Silakan anda gunakan component sesuai dengan kebutuhan anda.
  • Link gambar: https://s3-ap-southeast-1.amazonaws.com/niomic/img-v1/c_login_logo.png
  • Penggunaan component form, checkbox:
    <Form>
   <Form.Field>
     <label>First Name</label>
     <input placeholder='First Name' />
   </Form.Field>
   <Form.Field>
     <label>Last Name</label>
     <input placeholder='Last Name' />
   </Form.Field>
   <Form.Field>
     <Checkbox label='I agree to the Terms and Conditions' />
   </Form.Field>
   <Button type='submit'>Submit</Button>
 </Form>

Link dokumentasi component form : https://react.semantic-ui.com/collections/form/
