Props pada Class Component

Type data pada component react itu cuma dua yaitu Prop dan State.

  • Jika data berubah maka component berubah ( render ulang )
  • Ada 2 cara membuat component, functional component dan class component\
    • Stateless Component — tidak memiliki state hanya prop. Umumnya component ini di buat dengan functional component karena codenya lebih ringkas;
    • Statefull Component — memiliki state & props. Component ini dibuat dengan class. Kelebihan dari class component adalah memiliki lifecycle
  • Untuk sekarang ingat saja functional component tidak memiliki state, itu sebabnya barang ini biasa disebut stateless component.
  • Sedangkan class component bisa memiliki prop dan state.

Prop singkatan dari Property adalah sebuah nilai yang bisa diberikan terhadap suatu component, sehingga component tersebut bisa lebih dinamis.

  • Prop itu read-only.
  • Prop umumnya digunakan untuk komunikasi data component dari parent komponent ke child component.

Berikut kita akan membuat props pada component yang dibuat class:

  • Buat props pada component list.js:
    import React, {Component} from 'react';
import Image from './Image.js';

class List extends Component{
  render(){
    return(
      <div>
          <ol>
            <li>Satu</li>
            <Image sumberGambar="https://www.gstatic.com/webp/gallery3/1.png"/>
            <li>Dua</li>
            <Image sumberGambar="https://www.gstatic.com/webp/gallery3/2.png"/>
            <li>Tiga</li>
            <Image sumberGambar="https://www.gstatic.com/webp/gallery3/3.png"/>
            <li>Empat</li>
            <Image sumberGambar="https://www.gstatic.com/webp/gallery3/5.png"/>
          </ol>
      </div>
    );
  }
}

export default List;
  • Terapkan props pada class component image.js:
    import React, {Component} from 'react';

class Image extends Component{
  render(){
    return(
      <img
        src= {this.props.sumberGambar}
        alt="gambar-makanan"
        width="300"
        height="300"
      />
    );
  }
}

export default Image;

    Perubahan data pada props dan state akan merender ulang component.

    • Gunakan props untuk komunikasi data antar komponent ( parent to child component )
    • Gunakan state untuk komunikasi data internal, hanya dalam component itu sendiri
    • props dan state bisa memiliki default value
