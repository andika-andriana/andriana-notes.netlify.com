Penggunaan Component Native Base

NativeBase adalah sekumpulan React Native component yang dapat digunakan pada project react native kita. Asiknya disini satu kode bisa langsung digunakan di platform Android maupun iOS, berbeda dengan kepunyaan React Native yang masih ada beberapa component khusus tiap tiap platformnya.

NativeBase ini merupakan proyek sumber terbuka (Open Source) yang kode sumbernya dapat dijumpai pada repository github nativebase. Dari dokumentasi yang saya baca sekilas, sepertinya library ini cukup dapat diandalkan untuk mempercepat proses pembuatan aplikasi android menggunakan React Native.

Berikut contoh penggunaan component native base pada project React-Native:
