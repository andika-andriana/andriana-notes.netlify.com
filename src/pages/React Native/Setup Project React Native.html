Setup Project React Native

Berikut adalah contoh penerapan Setup React Native kedalam leptop/pc dengan os windows 64bit.

  1. Kunjungi situs resmi react native: https://facebook.github.io/react-native
  1. Terdapat dua cara untuk melakukan installasi react native:
    • Expo CLI Quickstart
    • React Native CLI Quickstart
  1. Pilih OS windows dan pilih target os yang ingin di buat, disini saya memilih Android:
  1. Selanjutnya tersedia ketereangan dependencies yang harus di install, disini saya akan menginstal Java SE Development Kit (JDK).
  1. Download Sesuaikan dengan OS yang digunakan:
  1. Jika proses download telah selesai, maka lakukan proses penginstallan JDK: https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
  1. Kemudian download Node JS dan lakukan penginstallan: https://nodejs.org/en/download/
  1. Download Android Studio : https://developer.android.com/studio/index.html
  1. Setelah proses download Selesai kemudian Install> Klik Next:
  1. Klik Next
  1. Agree
  1. Pilih Install:
  1. Tunggu proses penginstallan sampai selesai:
  1. Klik Next,
  1. Klik Finish
  1. Pilih OK
  1. Tunggu proses loading selesai.
  1. Pilih Don't Send
  1. Klik Next
  1. Pilih Custom installlation
  1. Sesuaikan UI theme
  1. Setup Configurasi apa saja yang di install:
  1. Klik Verify Setting dan klik finish
  1. Tunggu proses selesai:
  1. Setelah selesai, kemudian klik finish:
  1. Maka akan muncul jendela baru (Welcome):
  1. Kemudian Klik configure dan pilih SDK manager:
  1. Kemudian beralih ke menu seacrh pada windows dan cari Environtment Variable:
  1. Kemudian pilih Environtment VAR..
  1. Klik NEW
  1. Isikan nama variable:
  1. Kemudian isikan Value dari SDK manager:
  1. Kemudian paste dan klik oke.
  1. Kemudian pilih Edit pada Path kembali untuk config platfform-toolsnya:
  1. Kemudian klik new dan pastekan kembali lokasi SDK:
  1. Kemudian tambahkan \platform-tools dan klik ok:
  1. Untuk memastikan proses penginstallan dan setup sudah berhasil, maka kita lakukan pengecekan menggunakan terminal/cmd ketikan:
