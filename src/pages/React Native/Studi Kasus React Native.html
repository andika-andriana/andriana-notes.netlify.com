Studi Kasus - React Native

Pada sesi ini kita akan membuat aplikasi menggunakan framework native-base dengan react-native, Berikut adalah layout contoh aplikasi yang akan kita buat:

  • Load Project folder pada aplikasi text editor (contoh: Atom):
  • Jalankan ADV dari Android Studio:
  • Kemudian install fw native-base pada project kita, dengan cara membuka terminal/cmd kemudian ketikan kode:
    npm install native-base --save
  • Setelah selesai menginstall fw native-base kemudian install juga peer dependencies-Nya:
    react-native link
  • Jalankan project yang kita buat pada Android Virtual Device, dengan kode:
    react-native run-android
  • Buat 1 folder Component dan isi dengan 4 file js :
