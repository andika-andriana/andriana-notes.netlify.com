CREATE TABLE ... SELECT

Berbeda dengan query CREATE TABLE...LIKE yang hanya menduplikasinya strukturnya saja, query CREATE TABLE ... SELECT ini berfungsi untuk melakukan duplikasi tabel beserta nilai didalamnya;

Query ini pada dasarnya adalah gabungan dari 2 query yaitu create table dan select.

Format penulisan tabel ini adalah:

CREATE TABLE `TABEL BARU` SELECT `NAMA KOLOM` FROM `TABEL ASAL`;

Berikut contoh penerapannya:

MariaDB [indonesia]> SHOW TABLES;
+---------------------+
| Tables_in_indonesia |
+---------------------+
| 123456              |
| contoh_table        |
| dua                 |
| provinsi            |
| satu                |
+---------------------+
5 rows in set (0.001 sec)
