ALTER TABLE ... ADD COLUMN

ALTER TABLE … ADD COLUMN  

Query ALTER TABLE ... ADD COLUMN digunakan untuk menambah kolom baru ke dalam tabel.

Format dasar penulisannya adalah sebagai berikut:

ALTER TABLE nama_tabel
ADD [COLUMN] nama_kolom tipe_kolom [atribut_kolom] [FIRST | AFTER nama_kolom]

Kita akan bahas berbagai variasi dari query ALTER TABLE ... ADD COLUMN.

Perintah di dalam tanda kurung siku bersifat opsional dan boleh tidak ditulis. Sebagai contoh pertama, saya ingin menambah kolom c ke dalam tabel contoh_alter_1. Kolom c ini memiliki tipe data DECIMAL(4,2) dengan atribut NOT NULL. Berikut query-nya:

ALTER TABLE contoh_alter_1 ADD c DECIMAL(4,2) NOT NULL;


DESC contoh_alter_1;
+-------+--------------+------+-----+---------+-------+
| Field | Type         | Null | Key | Default | Extra |
+-------+--------------+------+-----+---------+-------+
| a     | int(11)      | YES  |     | NULL    |       |
| b     | varchar(10)  | YES  |     | NULL    |       |
| c     | decimal(4,2) | NO   |     | NULL    |       |
+-------+--------------+------+-----+---------+-------+

Namun kali ini kita akan menggunakan query after yang akan memposisikan kolom yang kita buat yaitu d setelah posisi b, contoh:

MariaDB [db_satu]> desc contoh_alter_1;
+-------+-------------------------------+------+-----+---------+-------+
| Field | Type                          | Null | Key | Default | Extra |
+-------+-------------------------------+------+-----+---------+-------+
| abc   | tinyint(4)                    | NO   | PRI | NULL    |       |
| b     | smallint(5) unsigned zerofill | YES  |     | NULL    |       |
| c     | decimal(4,2)                  | NO   |     | NULL    |       |
+-------+-------------------------------+------+-----+---------+-------+
3 rows in set (0.065 sec)

Contoh penggunaan query ALTER TABLE ... ADD COLUMN untuk menambah lebih dari satu kolom pada tabel:

MariaDB [db_satu]> ALTER TABLE contoh_alter_1
    -> ADD COLUMN f INT first,
    -> ADD COLUMN e INT after d;
Query OK, 0 rows affected (0.157 sec)
Records: 0  Duplicates: 0  Warnings: 0
