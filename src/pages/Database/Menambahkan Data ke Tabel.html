Menambahkan Data ke Tabel

Untuk menambahkan data ke table kita bisa menggunakan query insert into, dengan sitaks dasar sebagai berikut:

INSERT INTO <namaTable> VALUES <nilai_sesuai_urutan_kolom>

Berikut contoh penerapannya:

  1. Masuk kedalam mySQL kemudian pilih database yang akan kita gunakan;
    MariaDB [(none)]> show databases;
+--------------------+
| Database           |
+--------------------+
| indonesia          |
| information_schema |
| mysql              |
| performance_schema |
| phpmyadmin         |
| test               |
+--------------------+
6 rows in set (0.001 sec)

MariaDB [(none)]> use indonesia;
Database changed
MariaDB [indonesia]> show tables;
+---------------------+
| Tables_in_indonesia |
+---------------------+
| provinsi            |
+---------------------+
1 row in set (0.001 sec)
  1. (Opsional) melakukan pengecekan struktur table karena kita akan melakukan input sesuai urutan table yaitu default:
    MariaDB [indonesia]> desc provinsi;
+----------------+--------------+------+-----+---------+-------+
| Field          | Type         | Null | Key | Default | Extra |
+----------------+--------------+------+-----+---------+-------+
| id_prov        | int(11)      | YES  |     | NULL    |       |
| nama_prov      | varchar(100) | YES  |     | NULL    |       |
| ibu_kota       | varchar(100) | YES  |     | NULL    |       |
| populasi       | int(11)      | YES  |     | NULL    |       |
| tgl_diresmikan | date         | YES  |     | NULL    |       |
+----------------+--------------+------+-----+---------+-------+
5 rows in set (0.034 sec)
  1. Lakukan input menggunakan insert into;
    MariaDB [indonesia]> INSERT INTO provinsi VALUES
    -> (6,"Jawa Barat", "Bandung", 120238337,"2000-12-25");
Query OK, 1 row affected (0.052 sec)

    -> VALUES
    -> ("1999-05-05", 1999289, "Padang", "Sumatera Barat", 7);
