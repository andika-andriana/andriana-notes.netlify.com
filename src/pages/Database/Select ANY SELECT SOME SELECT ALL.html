Query SELECT...ANY, SELECT...SOME dan SELECT...ALL masih berhubungan dengan query SELECT...IN yang baru saja kita bahas. Ketiga query ini dipakai untuk operasi perbandingan dengan himpunan yang didapat dari hasil subquery (sebuah query didalam query lainnya).

Sebelum masuk ke penjelasan tentang ANY, SOME, dan ALL, saya akan kembali membahas query SELECT...IN yang melibatkan subquery. Perhatikan contoh berikut:

SELECT * FROM populasi WHERE kota IN (SELECT ibukota FROM daftar_provinsi);

Semuanya dipakai untuk membuat sebuah kondisi pada himpunan.
