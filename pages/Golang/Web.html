  1. Membuat Aplikasi Web Sederhanda

    PackageÂ net/httpÂ memiliki banyak sekali fungsi yang bisa dimanfaatkan. Di bagian ini kita akan mempelajari beberapa fungsi penting sepertiÂ routingÂ danÂ start server.

    Program dibawah ini merupakan contoh sederhana untuk memunculkan text di web ketika url tertentu diakses.

    package main

import "fmt"
import "net/http"

func index(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintln(w, "apa kabar!")
}

func main() {
    http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
        fmt.Fprintln(w, "halo!")
    })

    http.HandleFunc("/index", index)

    fmt.Println("starting web server at http://localhost:8080/")
    http.ListenAndServe(":8080", nil)
}

    Jalankan program tersebut.
