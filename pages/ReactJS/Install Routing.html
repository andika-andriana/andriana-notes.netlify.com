Install Routing

Disini kita hanya akan mempelajari bagaimana proses installasi react-router-dom pada react JS:

  • Pertama jalankan dulu npm kita;
  • Kedua ubah buatlah folder components yang didalamnya dibuat juga folder home;
  • Masukan file components kita kedalam folder components (disini hanya contoh file component project);
  • Kemudian import pada index.js;
  • Kemudian buka package.json dan buka terminal;
  • Kemudian install router dengan mengetikan kode berikut pada terminal:
    npm install --save react-router-dom
  • Jika telah berhasil terinstall maka react-router-dom akan muncul pada package.json;
