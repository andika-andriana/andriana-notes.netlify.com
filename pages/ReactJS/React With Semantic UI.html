React With Semantic UI

Semantic UI, framework CSS ini mempunyai banyak keunggulan seperti tampilan nya yang elegan, tidak berat saat di load pada browser, dapat di kustomisasi sesuai kemauan developer, memiliki beberapa tema yang dapat digunakan, dan juga mudah untuk di gunakan sebagai template pada website kamu.

Berikut cara penerapannya pada ReactJS:

  • Buat project react baru (contoh nama project: hello-semantic-ui):
    create-react-app hello-semantic-ui
  • Buka project tersebut pada text editor(contoh: atom.io):
  • Hapus file bawaan yang tidak digunakan:
  • Rapihkan kode:
  • Jalan kan npm untuk memastikan tidak ada yang salah:
  • Sebelum melakukan installasi framework semantic, kita cak dulu pada package.json untuk memastikan bahwa framework tersebut belum terinstall:
  • buka terminal baru pad folder project kita untuk menginstall semantic-ui:
  • Ketikan pada terminal tersebut (pastikan anda terhubung dengan internet):
    npm install --save semantic-ui-react semantic-ui-css
  • Tunggu prosesnya hingga selesai:
  • Untuk memastikan bahwa framework sudah berhasil terinstall cek pada package.json:
  • Mari kita coba terapkan contoh component pada App.js.
  • kemudian import semantic-ui
    import "semantic-ui-css/semantic.min.css";
import { Button } from "semantic-ui-react";
  • Test membuat button
    <Button primary>Button 1</Button>
<Button secondary>Button 2</Button>
  • Contoh pada App.js:
    import React from "react";
import "semantic-ui-css/semantic.min.css";
import { Button } from "semantic-ui-react";

function App() {
  return (
    <div>
      <h1>Hello Semantic UI</h1>
      <Button primary>Button 1</Button>
      <Button secondary>Button 2</Button>
    </div>
  );
}

export default App;
