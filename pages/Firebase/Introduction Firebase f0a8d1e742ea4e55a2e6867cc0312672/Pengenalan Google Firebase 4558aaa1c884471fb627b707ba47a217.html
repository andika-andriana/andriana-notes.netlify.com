Pengenalan Google Firebase

Pengenalan Firebase

Firebase adalah teknologi baas (back-end access service) yang disediakan oleh google untuk mempermudah developer dalam mengembangkan aplikasinya.

Sejarah Singkat Firebase:

  • Diciptakan:

    Firebase pertama kali didirikan pada tahun 2011 oleh Andrew Lee dan James Tamplin.

  • Produk Pertama:

    Produk yang pertama kali dikembangkan adalah realtime database. dimana developer dapat menyimpan dan melakukans sinkronisasi data ke banyak user.

  • Diakusisi Google:

    Pada 2014 Oktober, perusahaan Firebase tersebut diakusisi oleh Google, berbagai fitur terus dikembangkan hingga diperkenalkan pada Mei 2016 di Google I/0.

Layanan Firebase Populer

  1. Firebase Authentication
  1. Firebase Database
  1. Firebase Storage
  1. Firebase Hosting

Kelebihan Utama Firebase

  1. Tersedia versi free, sehingga para developer dapat mencoba layanan Firebase tanpa batas waktu.
  1. Cepat dan responsif.
  1. Tanpa SQL, Firebase menggunakan JSON.
  1. Powerful API untuk menyimpan dan sinkronisasi, API relatif mudah dimengerti dan realtime back-end.
  1. Memudahkan bagi para front-end developer yang kurang mengerti back-end.

Kekurangan Utama Firebase

  1. Jumlah koneksi node tidak ditampilkan (yang mana pada versi free dibatasi 100 node).
  1. Format penyimpanan tidak menggunakan SQL, melainkan menggunakan JSON sehingga akan menyulitkan jika akan melakukan migrasi.
  1. Karena Firebase tidak memberatkan developer pada server-side, maka konsekuensinya perlu kerja keras menggunakan API dalam kode kita di front-end/client side.
  1. Tidak ada fungsi reference ataupun join.
  1. You don't own your data, karena data kita berada dalam host bukan milik kita, akan menjadi sulit untuk merecovery akun user.
