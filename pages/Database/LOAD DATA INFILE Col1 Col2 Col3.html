LOAD DATA INFILE Col1, Col2, Col3 ...

Query ini berfungsi untuk menentukan kolom mana saja yang akan diisi didalam table.

Berikut contoh penerapannya:

  • Buat File.txt dengan nama data_provinsi dengan data seperti ini:
    Aceh,AC,Banda Aceh,4906835
Banten,BT,Serang,11704877
Bengkulu,BE,Bengkulu,1844800
Gorontalo,GO,Gorontalo,1115633
  • Kemudian buat table yang akan kita isikan data_provinsi:
    CREATE TABLE provinsi (
nama_prov VARCHAR(50),
kode_iso CHAR(2),
ibukota VARCHAR (50),
populasi INT,
luas DEC(8,2),
apbd DEC(8,2),
PRIMARY KEY(kode_iso)
);
