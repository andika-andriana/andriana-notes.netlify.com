Membuat API GET Data Laravel

Kali ini kita akan mempraktekan contoh membuat API GET Data pada Laravel, sebelumnya pastikan sudah terinstall laravel para project kita.

Fungsi API GET Data ini adalah untuk memanggil data yang akan ditampilkan dalam bentuk season, sebelum membuat fungsi tersebut kita harus melakukan configurasi pada databasesnya.

Langsung pada prakteknya, berikut adalah contoh pembuatan API GET pada framework laravel:

  1. Create databases pada xampp atau aplikasi lain pada local host dengan nama db_materi1
  1. Buka project kita dengan text editor, kemudian lakukan config pada file env
  1. Buat table dengan fitur migration dari php artisan yang tersedia di laravel
    PS C:\xampp\htdocs\hello-laravel> php artisan make:migration tbl_katalog
Created Migration: 2019_12_30_095023_tbl_katalog
  1. Buat struktur table pada database yang kita buat tadi:
    <?php

use Illuminate\Database\Schema\Blueprint;
use Illuminate\Database\Migrations\Migration;

class TblKatalog extends Migration
{
    /**
     * Run the migrations.
     *
     * @return void
     */
    public function up()
    {
      Schema::create('tbl_katalog', function(Blueprint $table){
        $table->increments('id');
        $table->string('nama_produk');
        $table->string('berat');
        $table->float('harga');
        $table->string('gambar');
        $table->string('keterangan');
      });
    }

    /**
     * Reverse the migrations.
     *
     * @return void
     */
    public function down()
    {
        //
    }
}
  1. Kemudian generate table tersebut dengan mengetikan kode pada terminal/cmd:
    PS C:\xampp\htdocs\hello-laravel> php artisan migrate
Migration table created successfully.
Migrated: 2014_10_12_000000_create_users_table
Migrated: 2014_10_12_100000_create_password_resets_table
Migrated: 2019_12_30_095023_tbl_katalog
  1. Jika berhasil, bisa kita lihat pada sqlyog/phpmyadmin bahwa table telah terbuat:
  1. Kemudian hapus table yang tidak kita gunakan (migrations, password_resets, users);
  1. Buat MVC Controller menggunakan php artisan;
    PS C:\xampp\htdocs\hello-laravel> php artisan make:controller Barang
Controller created successfully.
  1. Jika berhasil maka akan tampil file baru sesuai nama controller yang kita buat:
  1. Selanjutnya kita akan membuat function pada controller Barang dengan kode:
    <?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;
use Illuminate\Support\Facades\DB;
use App\Http\Requests;

class Barang extends Controller
{
  public function getData(){
    $data = DB::table('tbl_katalog')->get();
    if(count($data) > 0){
      $res['message'] = "Success";
      $res['value'] = $data;
      return response($res);
    }else{
      $res['message'] = "Empty";
      return response($res);
    }
  }
}
  1. Kemudian buat router pada routes.php:
    Route::get('/getData','Barang@getData');
  1. Kemudian jalankan server menggunakan kode :
    PS C:\xampp\htdocs\hello-laravel> php artisan serve
Laravel development server started on http://localhost:8000/
