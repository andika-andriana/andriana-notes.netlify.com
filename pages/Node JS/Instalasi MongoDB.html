Instalasi MongoDB

  1. Download file master MongoDB di website resminya di http://www.mongodb.org. Pada tutorial ini menggunakan OS Windows 10 64-bit.

    Gambar: Pilihan versi serta OS komputer

  1. Setelah file berhasil di download, klik kanan file tersebut lalu pilih run as administrator. Kemudian Next dan ceklist “I accept the terms in the License Agreement”.

    Gambar: License Agreement

  1. Pada Choose Setup Type pilih Complete, kemudian pilih Next dan isi service configuration seperti gambar dibawah ini.

    Gambar: Choose Setup Type dan Service Configuration

  1. Setelah itu pilih next untuk melanjutkan proses Ready to Install MongoDB dan tunggu sampai proses instalasi berhasil.

    Gambar: Proses Ready to install MongoDB

  1. Setelah proses instalasi selesai tekan tombol finish.

    Gambar: Proses Instalasi Selesai

  1. Sebelum menjalankan MongoDB, setting PATH agar mudah dipanggil saat membuka command prompt. Berikut ini cara melakukan setting PATH. Klik Kanan My Computer -> Properties, pilih Advance Klik Environment Variables, klik tombol edit kemudian isikan parameter berikut ini:

    Gambar: Edit Environment Variable

  1. Buka CMD kemudian ketik perintah mongo. Sekarang sudah masuk ke mongo terminal dan sudah bisa membuat database dengan mongodb.

    Gambar: MongoDB Terminal
